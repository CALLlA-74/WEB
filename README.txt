Backend запускается в файле main.py (директория Zlatoshvieka). При создании проекта использован framework Flask, а в качестве СУБД SQLite. 
БД находится по пути Zlatoshveika\database
Компоненты верстки находятся в директории Zlatoshveika\app\templates, а фотографии, стили, скрипты находятся в Zlatoshveika\app\static
